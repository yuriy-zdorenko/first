    Create a Repository

  • Create new local repository

    $ git init

  • Download from an existing repository

    $ git clone my-url

    Observe your repository

  • List new or uncommitted files

    $ git status

  • Show unstaged changes

    $ git diff

  • Show staged changes

    $ git diff ~~cached

  • Show all staged and unstaged changes

    $ git diff HEAD

  • Show the changes between two commit ID's

    $ git diff commit1 commit2

  • List the change dates and authors in file

    $ git blame [file]

  • Show a file change for a commit ID

    $ git show [commit] : [file]

  • Show full change history

    $ git log

  • Show change/history for field or directory including diffs

    $ git log -p [file/directory]

    Working with branches

  • Lists all branches in directory

    $ git branch

  • List all branches, local and remote

    $ git branch -av

  • Switch to a branch, and update working directory

    $ git checkout my_branch

  • Create a new branch

    $ git branch new_branch

  • Delete branches

    $ git branch -d my_branch

  • Merge branch_a into branch_b

    $ git checkout branch_b
    $ git merge branch_a

    Make a change

  • Stages file, ready for commit

    $ git add [file]

  • Stage all changed files, ready for commit

    $ git add .

  • Commit all staged files to versioned history

    $ git commit -m "commit message"

  • Commit all your tracked files to versioned history

    $ git commit -am "commit message"

  • Reset everything to the last commit

    $ git reset --hard

  • Unstages file, keeping the file changes

    $ git reset [file]

    Synchronize

  • Get the latests changes from origin (no merge)

    $ git fetch

  • Fetch the latests changes from origin and merge

    $ git pull

  • Fetch the latests changes from origin and rebase

    $ git pull --rebase

  • Push changes to origin

    $ git push

    Finally!

  • When in doubt, use git help

    $ git command --help
