Програмування

Програмування - це моє вікно в світ майбутнього, де код стає живим виразом ідеї. Кожен рядок коду - це крок до створення чогось нового, неймовірного та корисного. Коли я сідаю за комп'ютер, то відчуваю себе справжнім майстром цифрового мистецтва. Ця незвичайна здатність трансформувати ідеї y віртуальну реальність завжди залишає мене в захваті. Мова програмування - це мій пензель, a комп'ютер - моя полотно, на якому я творю.

Кожен проект для мене - це виклик, що дарує найбільше задоволення. Чи то розробка веб-сайту, додатку для смартфону чи вирішення складних алгоритмічних завдань, кожен крок в цьому шляху принесе велике задоволення та надихає на нові досягнення.

A ще програмування - це спільнота розумних i творчих людей, які завжди готові допомогти та поділитися своїми знаннями. Вивчення нових мов програмування та технологій - це завжди цікава подорож y світ інновацій та можливостей.

export const MainLayout: FC = () => {
  const theme = useAppSelector((store) => store.theme)
  const element = document.documentElement

  const darkQuery = window.matchMedia('(prefers-color-scheme: dark)')

  function onWindowMatch() {
    if (localStorage.theme === 'dark' || (!('theme' in localStorage) && darkQuery.matches)) {
      element.classList.add('dark')
    } else {
      element.classList.remove('dark')
    }
  }

  onWindowMatch()

  useEffect(() => {
    switch (theme.theme) {
      case 'Темна':
      case 'Dark':
        element.classList.add('dark')
        localStorage.setItem('theme', 'dark')
        break
      case 'Світла':
      case 'Light':
        element.classList.remove('dark')
        localStorage.setItem('theme', 'light')
        break
      default:
        localStorage.removeItem('theme')
        onWindowMatch()
        break
    }
  }, [theme])
