Від маленького віку я завжди був зацікавлений у правових аспектах життя, і ця цікавість росла разом зі мною. Я завжди вірив у справедливість та правовий порядок як основні складові нашого суспільства, і саме тому обрав цей шлях.

Університетське навчання допомагає мені розширювати мої знання і розвивати аналітичне мислення. Окрім теоретичних знань, я здобуваю практичні навички у різних сферах права, від цивільного до кримінального. Ця різноманітність допомагає мені зрозуміти, яка галузь юриспруденції мені найбільше близька, і яким напрямком я бажаю йти в майбутньому.

Моя мета - стати кваліфікованим та справедливим юристом, який зможе надавати юридичну допомогу тим, хто цього потребує. Я вірю, що правова освіта дає мені можливість впливати на подальший розвиток правопорядку та суспільства в цілому. І хоча навчання юриспруденції може бути складним і вимагати багато зусиль, я готовий пройти цей шлях, бо вірю в важливість нашої професії для будь-якого суспільства.

Музика дарує мені можливість виразити свої емоції та знайти внутрішній спокій, що особливо важливо в напруженому світі правознавства. Це моє вікно в інший світ, де правила встановлюються мелодією, а справедливість виражається в гармонії. Моя гітара - це не просто інструмент, вона - частина моєї ідентичності і джерело незабутніх моментів у моєму житті.

#include <iostream>
#include <algorithm>

int main()
{
    const int length(9);
    int array[length] = { 7, 5, 6, 4, 9, 8, 2, 1, 3 };

    for (int iteration = 0; iteration < length-1; ++iteration)
    {
        int endOfArrayIndex(length - iteration);

        bool swapped(false);

        for (int currentIndex = 0; currentIndex < endOfArrayIndex - 1; ++currentIndex)
        {
            if (array[currentIndex] > array[currentIndex + 1])
            {
                std::swap(array[currentIndex], array[currentIndex + 1]);
                swapped = true;
            }
        }

        if (!swapped)
        {
            std::cout << "Early termination on iteration: " << iteration+1 << '\n';
            break;
        }
    }

    for (int index = 0; index < length; ++index)
        std::cout << array[index] << ' ';

    return 0;
}
